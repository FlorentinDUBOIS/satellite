# Satellite

> Retrieve satellite over celestrack using time series

## Installation

This script need a python2 installation and some dependencies. Those dependencies can be install using pip.

Install script requirements:

```sh
$ pip install ephem requests
```

## Usage

Usage of satellite:

```sh
$ ./satellite.py --help

usage: satellite.py [-h] [-o OUTPUT]

optional arguments:
  -h, --help            show this help message and exit
  -o OUTPUT, --output OUTPUT
                        Set the output file
```

Launch the scraper:

```sh
$ ./satellite.py
```

This will create a `stations.metrics` file containing [sensision](http://www.warp10.io/getting-started/#data-format) metrics.
